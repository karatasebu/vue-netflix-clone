# Netflix Clone

Netflix Clone made with Vue.js and SCSS. Datas fetched from TMDB API.

## [Demo website](https://vue-netflix-clone-one.vercel.app/)

## Installation

```bash
git clone https://github.com/karatasebu/vue-netflix-clone.git
cd vue-netflix-clone
npm i
```

## Start the server

```bash
npm run dev
```
